# Contributing to Thales Open Quant Initiative

Thank you for your interest in contributing to **Thales Open Quant Initiative**!  
This document guides you on how to contribute effectively and collaboratively.

---

## **Ways to Contribute**
- Add a **new project or notebook**  
- Improve or extend **existing projects**  
- Fix **bugs or errors** in code or documentation  
- Add **visualizations, examples, or explanations**  
- Suggest **new learning resources** or references

---

## **Getting Started**
1. **Fork the repository**  
2. **Clone it locally:**
   ```bash
   git clone https://github.com/<your-username>/thales-open-quant.git
   cd thales-open-quant

python -m venv env
source env/bin/activate  # On Windows use `env\Scripts\activate`
